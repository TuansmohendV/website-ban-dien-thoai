DAH SACH API BACKEND DA LAM

Base URL mac dinh: http://localhost:8080

Ghi chu:
- Public: khong can dang nhap
- User: can Bearer token nguoi dung
- Admin: can Bearer token admin
- Optional Auth: cho phep user dang nhap hoac guest thong qua x-guest-id

1. HEALTH CHECK

GET /
- Quyen: Public
- Mo ta: Kiem tra server backend dang chay va liet ke cac module chinh

2. AUTH API

POST /api/auth/register
- Quyen: Public
- Mo ta: Dang ky tai khoan moi

POST /api/auth/login
- Quyen: Public
- Mo ta: Dang nhap bang email, so dien thoai hoac identifier

POST /api/auth/logout
- Quyen: User
- Mo ta: Dang xuat tai khoan

POST /api/auth/forgot-password
- Quyen: Public
- Mo ta: Tao yeu cau quen mat khau

POST /api/auth/reset-password
- Quyen: Public
- Mo ta: Dat lai mat khau bang token

PUT /api/auth/change-password
- Quyen: User
- Mo ta: Doi mat khau khi da dang nhap
